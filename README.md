# angular2-webpack-minimal-boilerplate
angular2 webpack typescript minimal boilerplate

## 0. Prerequisite
```
npm install -g webpack webpack-dev-server typings
```

## 1. Install
```
npm install
```

## 2. Development
```
npm run dev
```
open http://127.0.0.1:3000

## 3. Build
```
npm run build
```
